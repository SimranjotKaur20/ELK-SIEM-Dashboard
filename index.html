Understanding the Log Pipeline:

  1. Endpoint Monitoring: Log Shippers (Winlogbeat/Filebeat) sit on target servers, capturing raw audit and logon logs.
  2. Ingestion & Parsing: Shippers forward events to Logstash on Port 5044. Logstash dissects unstructured fields using regular expression Grok filters, normalizes variables to Elastic Common Schema (ECS), and appends geographical metadata to public IPs.
  3. Storage & Alerting: Logstash feeds parsed events to Elasticsearch. In Elasticsearch, Watcher rules continuously check incoming entries for anomalies (e.g. brute force triggers) and create notifications.
  4. Analytics Visualization: Kibana acts as a graphical facade to Elasticsearch data, performing queries and building operations control dashboard views.
