# ELK SIEM Dashboard

## 📌 Overview

The ELK SIEM Dashboard is a cybersecurity project that demonstrates centralized log management, real-time monitoring, and security event visualization using the ELK Stack (Elasticsearch, Logstash, and Kibana).

The dashboard collects logs, processes them through Logstash, stores them in Elasticsearch, and visualizes security events in Kibana.

---

## 🚀 Features

- Centralized log collection
- Real-time security monitoring
- Interactive Kibana dashboards
- Log parsing using Logstash
- Elasticsearch indexing
- Search and filtering
- Security event visualization
- Custom dashboards
- Sample log analysis
